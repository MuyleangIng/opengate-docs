# OAuth Clients API

Base path: `/api/realms/{realm}/clients`

## List Clients

```
GET /api/realms/{realm}/clients
```

## Register Client

```
POST /api/realms/{realm}/clients
```

<CodeBlock language="json" code={`{
  "realmName": "master",
  "publicClient": false,
  "pkceRequired": true,
  "redirectUris": ["http://localhost:3000/callback"],
  "webOrigins": ["http://localhost:3000"],
  "grantTypes": ["authorization_code", "refresh_token"],
  "scopes": ["openid", "profile", "email"]
}`} />

**Response:**
<CodeBlock language="json" code={`{
  "id": "550e8400-...",
  "clientId": "client-a1b2c3d4",
  "clientSecret": "cs_xxxxxxxxxxxxxxxx",
  "publicClient": false,
  "pkceRequired": true,
  "redirectUris": ["http://localhost:3000/callback"],
  "grantTypes": ["authorization_code", "refresh_token"],
  "scopes": ["openid", "profile", "email"]
}`} />

<Callout variant="warning" title="Save the client secret">
The `clientSecret` is only returned **once** on creation. Store it securely. Use the regenerate endpoint to rotate it.
</Callout>

## Regenerate Client Secret

```
POST /api/realms/{realm}/clients/{clientId}/secret/regenerate
```

**Response:**
<CodeBlock language="json" code={`{ "clientSecret": "cs_yyyyyyyyyyyyyy" }`} />

## Delete Client

```
DELETE /api/realms/{realm}/clients/{clientId}
```

## Client Types

| Type | `publicClient` | `pkceRequired` | Secret |
|------|----------------|----------------|--------|
| SPA / Mobile | `true` | `true` | None |
| Server-side app | `false` | `false` | Required |
| Machine-to-machine | `false` | `false` | Required |
