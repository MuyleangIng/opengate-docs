# Microservices

OpenGate IAM is composed of 10 microservices + 1 shared library.

## opengate-common

Shared library included by all services. Not a runnable service.

- `PageResponse<T>` — generic paginated response record
- `OpenGateException` / `ResourceNotFoundException` — typed exceptions
- `GlobalExceptionHandler` — `@RestControllerAdvice` for 404/400
- `DomainEvent` — Kafka event envelope with `eventId`, `eventType`, `aggregateId`, `realm`, `payload`, `occurredAt`
